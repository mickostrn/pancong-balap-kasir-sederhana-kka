# Kasir Pancong Balap

Aplikasi kasir sederhana berbasis Command Line Interface (CLI) menggunakan Python.

## Screenshot

![Tampilan Program](/screenshoot/program.png)

## Fitur

* Menampilkan daftar menu makanan dan minuman
* Filter menu berdasarkan kategori
* Mengubah jumlah pesanan
* Menambah pesanan
* Menghapus pesanan
* Menghitung total pembayaran
* Menghitung diskon
* Proses pembayaran
* Cetak struk transaksi

## Struktur Data

Data menu disimpan dalam sebuah list yang berisi beberapa dictionary seperti:
* Nama produk
* Harga
* Kategori
* Variasi

## Cara Menjalankan

* Pastikan Python sudah terpasang.
python kasir_pancong_balap.py
